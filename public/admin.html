    Apply update

    CLI install detected
    
        
    In-app update is only available for installs done via the browser flow. You installed this site via the terminal, so updates run the same way:
    
        
    curl -fsSL https://seo.benjaminb.xyz/install/run.sh | bash
    
        
    It detects an existing install with the same project slug and only re-deploys the code — your D1, R2, and admin user are preserved.

          How to get a service-account JSON (3 minutes)
          
        
            
      1. Open Google Cloud → Create project. Name it anything (e.g. "pages-seo-indexing").
        2. 
            
      2. In that project, open Search Console API and click Enable. Also enable the Indexing API if you want per-URL pings.
        3. 
            
      3. Go to Service accounts → Create service account. Skip the role + user steps.
        4. 
            
      4. Click the account → Keys tab → Add Key → JSON. Save the file.
        5. 
            
      5. In Search Console → Users and permissions, click Add user. Paste the service account email (it ends in iam.gserviceaccount.com). Set permission to Owner.
        6. 
            
      6. Open the JSON file in a text editor, copy the entire contents, paste below.
