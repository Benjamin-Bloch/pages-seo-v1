How to use it

  1. Pick a mode above (Install if you're new, Update to refresh an existing site, Repair if something's broken).
  2. Click "Copy prompt" — the full prompt copies to your clipboard.
  3. Open ChatGPT / Claude / Gemini (any of the three works equally well; pick the one you already have an account for).
  4. Paste and send. The AI will read the prompt and start asking you questions or giving instructions one step at a time.
  5. Follow along. Tell the AI when you've finished each step, and if anything errors out paste the error back to the AI — it knows the common failure modes and how to fix them.
